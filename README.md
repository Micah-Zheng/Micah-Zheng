我是偏工程落地的 JS 全栈开发者，习惯从产品体验、接口设计、数据结构和后续维护一起考虑问题。

和只会“把页面做出来”相比，我更在意这些事：

- 能不能把一个需求完整做通，而不是只完成界面层。
